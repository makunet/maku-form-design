# maku-form-design

> vue3 表单设计器

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run dev
```

### Compiles and minifies for production

```bash
npm run build
```

## Usage

```bash
git clone https://gitee.com/makunet/maku-form-design
cd maku-form-design
npm i
npm run dev
```
